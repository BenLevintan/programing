Library Management System
This project implements a simple library management system in Java, allowing users to manage books, borrow books, return books, and generate various reports.

Getting Started
These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

Prerequisites
To run this project, you need to have the following software installed on your machine:

Java Development Kit (JDK) 8 or higher
IntelliJ IDEA (recommended) or any other Java IDE
Installation
Clone the repository to your local machine:

git clone https://github.com/your-username/library-management-system.git

Open the project in IntelliJ IDEA or your preferred Java IDE.

Build and run the project.

Usage
Once the project is running, you can use the provided main class to test the functionality of the library management system. The main class contains various test cases to demonstrate different features of the system, such as adding books, borrowing books, returning books, and generating reports.

Features
Add books to the library with title and author information.
Borrow books from the library.
Return books to the library.
Generate reports, such as total books in the library, total available books, total loan books, and author with the most books.
